Projetos

Receitando

Apresentamos um projeto especial, meticulosamente elaborado para abordar o consumo avançado de APIs, com foco particular em receitas culinárias. Este projeto oferece uma variedade de recursos, incluindo a capacidade de pesquisar receitas, manipular ingredientes, criar receitas personalizadas e fazer upload diretamente na API. Além disso, os usuários podem favoritar suas receitas preferidas para acessá-las facilmente a qualquer momento. Tudo isso é apresentado em um template elegante e responsivo, garantindo uma experiência de usuário imersiva e agradável.

Sass, parcel, Js (ES6+ Moderno), API (forkify)
